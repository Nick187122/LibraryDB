# LibraryDB

## Project Description
A comprehensive relational database for managing library operations including:
- Member management
- Book cataloging
- Loan tracking
- Fine calculation
- Reservation system

## Database Features
- Proper normalization (3NF)
- Data integrity constraints
- Relationship management (1-1, 1-M, M-M)
- Performance optimization with indexes
